My Work

www.devify.live

Devify is a music playlist creator built with React.js, SQL and Node.js. It also stores the user session in Redux which creates a unique experience for the user. Users can upload songs using Amazon S3 to an open source library and add songs from that library to as many playlist as they like.

Auto e-commerce site

A group project built by myself and two other developers. This site allows users to browse cars that are for sale. If a user registers they can create a profile and post their own car for sale as well as get in contact with other sellers and buyers.

About
